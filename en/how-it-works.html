How It Works

A simple 4-step process to implement environmental monitoring in your school

1. Schools Order

Get complete kits with all necessary components, sensors, and hardware, or just download the detailed plans and source components yourself. Both options include comprehensive setup guides and educational materials.

2. Students Build

Students assemble the air quality monitoring station at school through hands-on workshops, learning electronics, programming, and environmental science. Our guides support both classroom and remote learning scenarios.

3. Stations Deployed

Completed stations are installed around the school campus to monitor air quality, temperature, and humidity in real-time, protecting the school community with timely alerts and actionable data.

4. Data Collected

Environmental data is automatically stored in global databases via our secure API, enabling students to analyze trends, contribute to climate research, and compare data with schools worldwide.

Data & API Access

Access real-time and historical air quality data through our API

API Endpoints

GET /api/v1/schools

List all participating schools

GET /api/v1/aqi/{region}

Get regional air quality data

GET /api/v1/historical/{school_id}

Historical data for specific school

Integration Example

curl -X GET https://api.schoolair.org/v1/aqi/europe \ -H "Authorization: Bearer YOUR_API_KEY" # Response: { "region": "europe", "avg_aqi": 45, "schools": 692 }
