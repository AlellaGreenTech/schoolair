Project Features

Everything you need for successful environmental monitoring and education

🌡️ Weather Station

ENV IV Unit: Measures temperature, humidity, and barometric pressure with high precision for comprehensive weather monitoring.

🌫️ Air Quality Monitor

PM2.5 Sensor: Detects and quantifies PM1, PM2.5, and PM10 particulate matter for critical air pollution assessment.

📡 IoT Connectivity

M5Stack CoreS3: Wi-Fi enabled microcontroller with built-in display for real-time data visualization and cloud connectivity.

☁️ Cloud Database

PostgreSQL Storage: Secure data transmission via Node-RED API to persistent cloud database for historical analysis.

📊 Data Visualization

Data dashboards: Interactive, real-time dashboards for monitoring trends, patterns, and environmental changes.

🎓 Curriculum Integration

STEM Learning: Comprehensive guides for integrating the project into science, math, technology, and social studies curricula.

Technology and Components

The goal of the SchoolAIR project is to support all the most popular educational microcontrollers, such as Arduino, m5stack, and others. Student volunteers are constantly porting the base software to new platforms. Communications between the microcontroller and sensors are done via standard protocols like I2C, Bluetooth, and WiFi. This keeps costs to a minimum and creates an open system. SchoolAIR is for students and by students!
