Endpoints de API

GET /api/v1/schools

Listar todas las escuelas participantes

GET /api/v1/aqi/{región}

Obtener datos regionales de calidad del aire

GET /api/v1/historical/{school_id}

Datos históricos para escuela específica

Ejemplo de Integración

curl -X GET https://api.schoolair.org/v1/aqi/europe \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Accept: application/json" # Respuesta: { "region": "europe", "avg_aqi": 45, "schools": 692, "last_updated": "2024-01-15T10:30:00Z" }
