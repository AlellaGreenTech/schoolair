How It Works

From sensors to cloud: understanding your environmental monitoring station

1

Data Collection

M5Stack CoreS3 reads sensor data: temperature, humidity, pressure, and particulate matter concentrations.

2

Secure Transmission

Data sent via Wi-Fi to Node-RED backend API using secure HTTP POST requests with API key authentication.

3

Cloud Storage

PostgreSQL database stores all environmental data for long-term historical tracking and analysis.

4

Visualization

Grafana dashboards provide real-time monitoring, trend analysis, and educational data exploration tools.
